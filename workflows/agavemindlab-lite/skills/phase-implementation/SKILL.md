# Implementation

Goal: produce a reviewable PR that satisfies the approved direction.

Do:
- Read `AGENTS.md` or equivalent project instructions before editing.
- Make the smallest correct change; reuse existing code and commands.
- Run scope-appropriate validation and record exact commands.
- Commit with `symphony-commit` when needed, then publish/update the PR with `symphony-pr`.
- Publish `## Implementation` with conclusion, PR link, files changed, validation, and material risks.

Exit after the artifact is posted for human review. Do not merge from this phase.
